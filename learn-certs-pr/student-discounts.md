# Descuentos para estudiantes

## Acerca de los descuentos para estudiantes

Nos complace ofrecer precios académicos en los exámenes de Microsoft Certification en la mayoría de los países/regiones (excepto India y China). Para poder beneficiarte del descuento para estudiantes, primero debes identificarte como estudiante en tu perfil Microsoft Certification y, a continuación, verificar tu estatus académico. Una vez verificada, podrás recibir el descuento de estudiante al inscribirte en un examen.

## <a name="how-to-identify-yourself-as-student-in-profile"></a> Cómo identificarte como estudiante en tu perfil de Microsoft Certification

1. Inicia sesión en tu [Microsoft Certification Dashboard](https://aka.ms/certdashboard) con tus credenciales de cuenta de Microsoft.
2. Selecciona "Editar tu perfil de certificación" en la parte superior de la página.
<br/>
<br/>
![Selecciona editar tu perfil de certificaciones](images/student-discounts-1.png)
<br/>
<br/>

3. Para editar tu perfil, selecciona el icono del lápiz junto a "Perfil de certificaciones".
<br/>
<br/>
![Selecciona el icono del lápiz para editar tu perfil](images/student-discounts-2.png)
<br/>
<br/>

4. En el menú desplegable "Job title", selecciona “Student”.
<br/>
<br/>
![Selecciona estudiante en el menú desplegable del título del trabajo](images/student-discounts-3.png)
<br/>
<br/>

5. Busca el aviso de precios académicos que aparece debajo del menú desplegable "Job title".
<br/>
<br/>
![Selecciona estudiante en el menú desplegable del título del trabajo](images/student-discounts-4.png)
<br/>
<br/>

## <a name="how-to-verify-your-academic-status"></a> Cómo verificar tu situación académica

1. Sigue las instrucciones para identificarte como estudiante en tu perfil de Microsoft Certification.
2. Busca el aviso de precios académicos que aparece debajo del desplegable "Job title". Selecciona "Get verified now" para ser redirigido al sistema de verificación académica.
<br/>
<br/>
![Selecciona Verificar ahora](images/student-discounts-5.png)
<br/>
<br/>

3. Puedes verificar tu situación académica con cualquiera de los métodos que se indican a continuación. Selecciona el método que prefieras en el menú principal. Sigue las instrucciones en pantalla para el método de verificación que hayas elegido.
    - Cuenta de correo electrónico emitida por la escuela.
    - Credenciales de la red de la escuela.
    - Tarjeta de identidad de estudiante internacional (ISIC).
    - Código de verificación de un representante de Microsoft o del administrador de tu institución.
    - Documentación (la documentación aceptable es un ID de estudiante con fecha, un informe de progreso actual, un horario de clases con fecha actual o una carta de aceptación en la escuela de educación superior).
<br/>
<br/>
![Verificar la situación académica](images/student-discounts-6.png)
<br/>
<br/>

## Comprobación de tu condición de estudiante al inscribirte en un examen

1. En la página de detalles de la certificación o del examen en el que quieras inscribirte, haz clic en "Programar examen".
2. Inicia sesión con las credenciales de tu cuenta de Microsoft asociadas a tu perfil de Microsoft Certification.
3. Cuando verifiques que tu perfil de certificación está actualizado, asegúrate de que el campo "Job title" muestra "Student-verified".
4. Si tu perfil de certificación no muestra este estado, sigue las instrucciones de [Cómo identificarte como estudiante en tu perfil de Microsoft Certification](#how-to-identify-yourself-as-student-in-profile) o [Cómo verificar tu estado académico](#how-to-verify-your-academic-status).
